## Build Setup
```bash
#install react-native first
$ npm i react-native -g
# git clone
$ git clone https://github.com/laclys/github_show.git
# cd project
$ cd github_show
# install dependencies
$ npm install
# run
$ react-native run-ios/run-android
```
